# 🚀 GUIA RÁPIDO - CONFIGURAR SECRETS EM 5 MINUTOS

## ⚡ **PASSO 1: GitHub Settings**
```
GitHub → Settings → Secrets and variables → Actions → New repository secret
```

## ⚡ **PASSO 2: Configurar as 5 Secrets**

### **1️⃣ SMTP_SERVER**
```
Name: SMTP_SERVER
Value: smtp.gmail.com
```

### **2️⃣ SMTP_PORT**
```
Name: SMTP_PORT
Value: 587
```

### **3️⃣ SMTP_USERNAME**
```

```

### **4️⃣ SMTP_PASSWORD**
